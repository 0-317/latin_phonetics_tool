Инструмент для разделения слогов латинского текста

1. Общее представление о проекте

Это инструмент для обработки латинского текста. Он предназначен для помощи пользователям (особенно студентам, изучающим латынь) в понимании и освоении правил деления латинских слов на слоги и постановки ударения, что способствует правильному чтению и пониманию текста.

2. Функции каждого файла

1 app.py (точка входа в приложение)

Это основной центр запуска и управления всем веб-приложением.

2 project/hexingongneng.py (ядровые функции)

Это модуль с основной логикой проекта, который отвечает за всю языковую обработку.

3 templates/index.html (пользовательский интерфейс)

Это графический интерфейс, через который пользователь взаимодействует с приложением.

3. Направления для дальнейшего расширения

В дальнейшем можно добавить анализ стиха для поэзии, добавить возможность загрузки файлов или внедрить базу данных для сохранения и запроса  записей.
